# 🧠 ClearMind

AI productivity assistant that transforms chaos into clarity.

## Features
- Brain dump analysis with Groq LLM
- Notes, Tasks, History management
- Dark/Light mode
- Mobile responsive

## Tech Stack
- Next.js 15
- Groq API
- shadcn/ui
- Tailwind CSS

## Getting Started
```bash
npm install
npm run dev
```

Set `GROQ_API_KEY` in `.env.local`

## Deploy
Deploy to Vercel with one click.
